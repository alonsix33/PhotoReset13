# stickers/

Los stickers (bola 8, sello "13" en Creepster, corazón pixel) se dibujan
**vectorialmente en el canvas** al componer el PNG final — ver
`src/lib/stickers.ts`. No hay bitmaps aquí a propósito: dibujarlos por código
garantiza nitidez a 1200×1800 @300dpi.

Si en el futuro se quiere hornear un sticker como imagen, debe entrar aquí en
**alta resolución o SVG** (a 300dpi cualquier pieza chica sale suave impresa).
